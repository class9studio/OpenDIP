# 2.3 灰度变换

| process type | description                                               | done or not | 备注 |
| :----------- | :-------------------------------------------------------- | :---------- | :--- |
| 图像反转     | 灰度反转，尤其用与医疗图像                                |             |      |
| 对数变换     | 对数函数变换灰度，拉高低灰度，<br/>抑制高灰度，增强对比度 |             |      |
| 分段函数拉伸 | 对比度拉伸，增强细节                                      |             |      |
| 直方图统计   | 统计各灰度值分布                                          |             |      |
| 直方图均衡化 | 根据灰度概率分布拉伸像素值在指定灰度级的分布              |             |      |
